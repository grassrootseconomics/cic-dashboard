# Introduction

This docs repo contains all necessary info to configure, deploy and backup/restore the `CIC Stack` in **production**.

## Prerequisites

### Environment

- [`Kubernetes`](https://kubectl.docs.kubernetes.io/installation/kubectl/binaries/)
- [`Docker`](https://github.com/docker/docker-install)

### Other CLI tools

- [`doctl`](https://github.com/digitalocean/doctl)
- [`helmsman`](https://github.com/Praqma/helmsman)
